== Description ==

Integrates your WordPress blog to use e-commerce provided by Pagamento Digital.

== Installation ==

How to install Pagamento Digital WP e-Commerce

e.g.

1. Upload `pagamento-digital-wp-e-commerce` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the plugin through the 'Pagamento Digital' menu in WordPress
